Date

A JavaScript date is fundamentally specified as the number of milliseconds that have elapsed since the ECMAScript epoch, which is defined as the midnight at the beginning of January 1, 1970, UTC (equivalent to the UNIX epoch).

Constructor

Date()

When called as a function, returns a string representation of the current date and time. All arguments are ignored. The result is the same as executing new Date().toString().

new Date()

When called as a constructor, returns a new Date object.
